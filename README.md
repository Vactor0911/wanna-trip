## 개발 배경

> 여행은 가고 싶은데 계획을 짜기는 귀찮아

여행을 떠나는 것은 늘 신나고 설레는 일입니다.  
ㅤ하지만 계획을 짜고 문서로 정리하는 일도 그럴까요?  

<br />

많은 사람들이 여행을 계획하며 인터넷에서 정보를 찾고 동선을 정리하는 과정에서 피로감을 호소하고 있습니다.  
ㅤ인터넷 곳곳에 흩어진 여행 정보를 모으고 이를 다시 엑셀이나 메모장, SNS 메신저를 오가며 정리하는 과정은 너무 번거롭죠.  
ㅤ여행 계획을 짜다가 진이 다 빠져버린 경험, 한 번쯤 경험해 보신 적 있으실 겁니다.

<br />

만약 흩어져있는 여행 정보를 한 곳에서 볼 수 있다면 좋지 않을까요?  
ㅤ엑셀이나 파워포인트, 메모장 여기저기 옮겨 다니지 않고 한 곳에서 모든 걸 정리할 수 있다면요?  
ㅤ또, 여행 계획 짜기가 너무 귀찮은 나를 위해 누군가 대신 계획을 모두 만들어 주는 건 어떤가요?

<br />

`"아, 여행 계획 좀 편하게 만들고 싶다"`   
ㅤ이런 생각, 한 번쯤 해보신 적 없으신가요?

<br />

그런 여러분을 위해 **여행갈래**가 등장했습니다.
