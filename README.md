# Wanna Trip - 여행 계획 웹 사이트

[여행갈래? 백엔드 리포지토리](https://github.com/Vactor0911/wanna-trip-server)

## 1. 작품 개요
- 여행갈래(Wanna Trip)는 유튜브·블로그·검색 등 여러 출처에 흩어진 여행 정보를 일일이 모아 정리해야 하는 번거로움과 서로 다른 양식 때문에 계획을 통합·수정하기 어려운 문제를 해결하고자 함
- 한 플랫폼에서 정보 집약과 커뮤니티 공유 계획의 즉시 가져오기 기능을 제공하고, AI로 초기 템플릿을 자동 생성해 사용자가 바로 편집·확장할 수 있게 하여 계획 수립의 시간과 노력을 크게 줄임

## 2. 작품 목표

- 분산된 출처의 여행 정보를 한곳에 집약하여 수집·정리에 드는 수작업을 대폭 감소 (목표: 최대 70% 절감)
- 커뮤니티에서 공유된 계획을 즉시 가져오고 템플릿화하여 계획의 재사용성과 통일성을 확보
- AI로 초기 템플릿을 자동 생성하여 누구나 빠르게 완성도 높은 일정을 만들 수 있게 함
- 실시간 협업 편집 및 지도·내보내기 연동으로 기획부터 실행까지 원스톱으로 지원
- 체계적인 여행 정보 관리와 공유 문화 확산을 통해 국내 여행 활성화 및 관광 산업 발전에 기여

## 3. 작품 설명 - 핵심 기능

### 3.1 사용자 인증 및 계정 관리
- **일반 로그인/회원가입**: 이메일 기반 계정 생성 및 JWT 토큰 인증
- **소셜 로그인**: 카카오, 구글 간편 로그인 지원
- **계정 연동**: 일반 계정과 소셜 계정 상호 연동 기능
- **이메일 인증**: 회원가입 시 이메일 인증 코드 발송 및 확인
- **비밀번호 관리**: 비밀번호 찾기, 재설정, 변경 기능
- **프로필 관리**: 닉네임 변경, 프로필 이미지 업로드, 계정 삭제
- **토큰 갱신**: Access Token 만료 시 Refresh Token으로 자동 갱신

### 3.2 AI 기반 여행 템플릿 생성
- **AI 자동 일정 생성**: 여행 정보 입력 시 AI가 초기 템플릿 자동 생성
- **맞춤형 추천**: 사용자 입력 기반 관광지·맛집·일정 최적화 추천
- **구조화된 데이터**: JSON 형식으로 일정 데이터 저장 및 관리
- **템플릿 미리보기**: 생성된 템플릿을 즉시 확인하고 수정 가능
- **빠른 편집**: AI가 생성한 기본 틀을 사용자가 자유롭게 확장·편집

### 3.3 여행 계획 관리 (Board & Card 시스템)
#### 📋 Board (일차별 관리)
- **일차 추가**: 여행 기간에 맞춰 Day 1, Day 2 등 보드 생성
- **드래그 앤 드롭**: 보드 순서를 자유롭게 변경
- **보드 복제**: 유사한 일정을 빠르게 복사
- **일괄 삭제**: 보드 내 모든 카드를 한 번에 삭제
- **카드 정렬**: 시간순, 수동 정렬 등 다양한 정렬 옵션

#### 🎴 Card (세부 일정 관리)
- **카드 생성/편집**: 세부 일정을 카드 단위로 작성
- **시간 설정**: 시작/종료 시간 지정 및 시간 중복 경고
- **카드 잠금**: 시간 중복 방지를 위한 잠금 기능
- **리치 텍스트 에디터**: 텍스트 서식, 이미지, 링크 삽입 지원
- **위치 정보 연동**: 네이버 지도에서 장소 검색 후 카드에 연결
- **카드 이동**: 보드 간 이동, 순서 변경 자유롭게 가능
- **카드 복제**: 유사한 일정 빠르게 복사

### 3.4 네이버 지도 통합
- **장소 검색**: 키워드로 관광지, 맛집, 숙소 등 검색
- **지도 시각화**: 선택한 장소를 지도에 마커로 표시
- **위치 정보 저장**: 장소명, 주소, 좌표, 카테고리, 썸네일 저장
- **전체화면 지도**: 큰 화면에서 지도 확인 및 장소 탐색
- **정적 지도 미리보기**: 카드에서 장소 위치 간단히 확인
- **경로 계획**: 일차별 방문 장소 순서 및 이동 경로 파악

### 3.5 커뮤니티 기능
- **게시글 작성/수정/삭제**: 여행 후기, 팁, 추천 일정 공유
- **리치 텍스트 에디터**: 이미지, 링크, 서식을 포함한 풍부한 콘텐츠 작성
- **게시글 조회**: 페이지네이션으로 게시글 목록 탐색
- **인기 게시글**: 좋아요 수 기반 인기 게시글 추천
- **인기 태그**: 자주 사용되는 태그 확인
- **좋아요 시스템**: 마음에 드는 게시글에 좋아요 표시
- **댓글 작성/수정/삭제**: 게시글에 댓글로 소통
- **타 사용자 템플릿 가져오기**: 커뮤니티에서 공유된 여행 계획을 내 템플릿으로 즉시 복사 **(핵심 강점)**
- **검색 기능**: 키워드로 템플릿 및 게시글 통합 검색

### 3.6 템플릿 관리 및 공유
- **나의 템플릿 목록**: 내가 작성한 템플릿 한눈에 보기
- **템플릿 편집/삭제**: 언제든 수정 및 삭제 가능
- **인기 템플릿 조회**: 다른 사용자가 많이 사용한 템플릿 확인
- **UUID 기반 공유**: 고유 링크로 템플릿 공유
- **템플릿 복제**: 기존 템플릿을 복사해 새로운 계획 생성

### 3.7 데이터 내보내기
- **엑셀 다운로드**: 여행 일정을 Excel 파일(.xlsx)로 내보내기
- **PDF 출력**: 일정표를 PDF 문서로 변환 및 저장
- **텍스트 내보내기**: 간단한 텍스트 형식으로 일정 저장
- **외부 공유**: 내보낸 파일을 이메일, 메신저 등으로 공유

### 3.8 보안 및 안정성
- **CSRF 보호**: Cross-Site Request Forgery 공격 방어
- **Rate Limiting**: API 호출 횟수 제한으로 서버 과부하 방지
- **Helmet.js**: HTTP 보안 헤더 설정으로 보안 강화
- **JWT 인증**: Access Token + Refresh Token 이중 인증 체계
- **쿠키 보안**: HttpOnly, Secure 쿠키로 토큰 안전하게 저장
- **환경변수 관리**: 민감한 정보를 환경변수로 분리 관리

## 5. 기대 효과 및 활용 방안

### 기대 효과

- **효율성 향상**: AI 기반 자동 일정 생성으로 여행 계획 시간 대폭 단축
- **접근성 개선**: 초보자도 직관적인 템플릿 시스템으로 전문가 수준의 일정 수립 가능
- **협업 강화**: 실시간 동시 작업으로 그룹 여행 계획의 효율성 증대
- **정보 공유 활성화**: 커뮤니티를 통한 여행 경험 및 노하우 축적으로 더 나은 의사결정 지원

### 활용 방안

- **개인 여행자**: 배낭여행, 자유여행 등 개인 일정의 체계적이고 효율적인 관리
- **가족/친구 여행**: 여러 구성원이 함께 계획을 수립하고 PDF/엑셀로 공유하여 협업
- **여행사 및 가이드**: 고객 맞춤형 여행 상품 기획 도구로 활용하여 빠른 견적 제시
- **교육 기관**: 수학여행, 현장학습 계획 수립 시 교사와 학생의 협업 도구
- **여행 블로거/인플루언서**: 커뮤니티에 템플릿 공유로 팔로워에게 가치 제공 및 콘텐츠 제작 활용
